# ابدأ من هنا — DZ-Gene (نسخة Rust)

## ما هذا؟

هذا **ليس تطبيقًا يعمل**. لا نافذة، لا أزرار، لا قوائم.

هذه هي **النواة العلمية** فقط: الكود الذي يقرأ ملفات `.edi`، وينسخ ويترجم
التسلسلات، ويحسب المسافات التطورية — مكتوبًا بلغة Rust.

## لماذا نبدأ بهذا وليس بالواجهة؟

لأن أخطر شيء في أي هجرة هو أن يتوقف بنك التسلسلات القديم عن الفتح.

فقبل أن نبني نافذة واحدة، نتأكد أولًا أن Rust يقرأ **ملفاتك الحقيقية** بالضبط
كما تقرؤها نسخة Python اليوم. إن نجح هذا، صار البناء فوقه آمنًا.

## الخطوات

### 1. ثبّت Rust (مرة واحدة فقط)

**ويندوز:**

1. افتح: <https://rustup.rs>
2. حمّل `rustup-init.exe` وشغّله.
3. سيسألك؛ اضغط `1` ثم `Enter` (الخيار الافتراضي).
4. **أغلق نافذة الطرفية وافتح واحدة جديدة** (ضروري).

**لينكس / ماك:**

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 2. تأكد أن التثبيت نجح

افتح طرفية (`cmd` أو PowerShell) واكتب:

```bash
cargo --version
```

يجب أن ترى شيئًا مثل `cargo 1.8x.x`. إن ظهر «الأمر غير معروف»، أعد فتح الطرفية.

### 3. فكّ ضغط المشروع

فكّ ضغط `dzgene-rs.zip` في مكان تختاره، مثلًا:

```
C:\dev\dzgene-rs\
```

**مهم:** حافظ على بنية المجلدات كما هي. يجب أن ترى:

```
dzgene-rs\
├── Cargo.toml
├── DECISION.md
├── crates\dzgene-core\
│   ├── Cargo.toml
│   ├── src\        (genetics.rs, edi.rs, distances.rs, lib.rs)
│   └── tests\      (banque_dz.rs + مجلد fixtures)
└── src-tauri\
```

### 4. شغّل الاختبارات

في الطرفية:

```bash
cd C:\dev\dzgene-rs
cargo test -p dzgene-core
```

أول مرة سيستغرق دقيقة أو دقيقتين (يحمّل ويصرّف المكتبات).

## ماذا ستتوقع؟

### الحالة (أ) — كل شيء أخضر

```
test result: ok. 38 passed; 0 failed
```

معناه: Rust يقرأ بنكك الحقيقي بشكل سليم، والترجمة والمسافات صحيحة.
**الأساس متين، ننتقل إلى المرحلة 1 (واجهة Tauri + العارض ثلاثي الأبعاد).**

### الحالة (ب) — أخطاء تصريف (`error[E0xxx]`)

**هذا متوقّع تمامًا، ولا يعني أن شيئًا خاطئًا في جهازك.**

كتبتُ هذا الكود في بيئة لم أستطع فيها تثبيت Rust (كان محجوبًا)، فلم أُصرّفه ولو
مرة واحدة. أخطاء التصريف في أول تشغيل أمر عادي.

**ما تفعله:** انسخ رسائل الخطأ كاملة وأرسلها لي. أُصلحها ونعيد المحاولة.

### الحالة (ج) — يصرّف لكن اختبار يفشل (`FAILED`)

هذه هي الحالة **المفيدة علميًا**: معناه أن Rust يقرأ ملفًا من بنكك بشكل مختلف
عن Python. أرسل لي اسم الاختبار الفاشل ورسالته — هذا بالضبط ما بُنيت الاختبارات
لكشفه.
